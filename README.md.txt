#  500W Simple Inverter using NE555 Timer IC

##  Overview
This project demonstrates the design and implementation of a 500W inverter system that converts 12V DC into 220V AC using basic electronic components.

##  Working Principle
- NE555 Timer generates a 50Hz square wave
- BC547 transistor amplifies the signal
- IRFZ44 MOSFETs perform switching (Push-Pull)
- Transformer converts 12V DC to 220V AC

##  Components Used
- NE555 Timer IC
- BC547 Transistor
- IRFZ44 MOSFETs (8)
- 12-0-12 Transformer
- Capacitors (1µF, 1000µF)
- Resistors
- 12V Battery

##  Output
- Input: 12V DC
- Output: ~220V AC (Square Wave)

##  Applications
- Power backup during outages
- Rural electricity solutions
- Educational project
